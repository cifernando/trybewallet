# TrybeWallet

An expense manager app that lets you add amounts, descriptions, and convert currencies in real time using the [AwesomeAPI](https://docs.awesomeapi.com.br/) exchange rate API.

Live app: https://trybewallet-taupe.vercel.app/

![App demo](https://user-images.githubusercontent.com/84880811/140509819-eb678480-fff2-43b1-ab4b-11959c97c1eb.gif)

## What I learned

- Create a Redux store in a React application
- Write reducers, actions, and dispatchers with Redux
- Connect Redux to React components
- Handle async actions in Redux (API calls)

**Status: Completed**

## Getting Started

```bash
git clone git@github.com:cifernando/trybewallet.git
cd trybewallet
npm install
npm start
```
